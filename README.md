# E-Ticketing Helpdesk

## Deskripsi

E-Ticketing Helpdesk merupakan aplikasi Flutter untuk membantu proses pelaporan, pengelolaan, dan monitoring tiket bantuan (Helpdesk). Sistem ini menyediakan tiga jenis pengguna yaitu Admin, Helpdesk, dan User.

---

## Fitur

- Login Multi Role
- Dashboard
- Manajemen Tiket
- Detail Tiket
- Tracking Status Tiket
- Komentar Tiket
- Notifikasi
- Profile User
- Reset Password
- Registrasi User
- Statistik Tiket

---

## Teknologi

- Flutter
- Riverpod
- Dio
- Flutter Secure Storage
- REST API (PHP Native)
- MySQL

---

## Cara Menjalankan

```bash
flutter pub get
flutter run
```
